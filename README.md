# DevOpsNotes

DevOpsNotes is a full-stack demo application showcasing modern DevOps practices through a simple CRUD mini-blog for technical notes.

## Key Features
- **MERN Stack**: MongoDB, Express/Node.js REST API, React frontend
- **Core Functionality**: CRUD articles with tags and slugs
- **DevOps Pipeline**: Dockerized, CI/CD-ready
